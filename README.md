✨ **Built a Tic Tac Toe Game using Python & Tkinter** 💻
<br>
**Highlights:**<br>
✦ Modern dark-themed GUI<br>
✦ Two Player Mode<br>
✦ Computer Mode (Easy AI)<br>
✦ Unbeatable AI powered by the **Minimax Algorithm**<br>
✦ Live Score Tracking<br>
✦ Winner Highlighting<br>
✦ New Round & Reset Match Options<br>
✦ Clean, interactive, and user-friendly interface <br><br>

## 🎮 Game Modes <br>

### 👥 Two Player Mode<br>
Play against another player on the same computer. <br>

### 🤖 Easy AI <br>
Play against a computer that makes random moves. <br>

### 🧠 Unbeatable AI <br>
Challenge an AI powered by the **Minimax Algorithm**, making it impossible to defeat with optimal play.<br><br>
